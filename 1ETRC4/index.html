UNIT-1 ( Discrete electronic devices )

TOPICS:- Diode, zener diode, BJT (Bipolar junction transistor), LED, photodiode, phototransistor, varactor; characteristic and operation (qualitative description and quantitative behaviour with black box approach)

UNIT-2 ( Diode circuits )

TOPICS:- clipper, clamper circuits, DC power supply: rectifier-half wave, full wave (center tapped, bridge), regulation (with regulator IC-LM317)

UNIT-3 ( BJT characteristics )

TOPICS:- ; BJT biasing; CE-biasing circuits: operating point; h-parameter model of transistor; large/small signal models (concept);large/small signal models of CE-BJT amplifier, Design of amplifier; Differential amplifier (using BJT).

UNIT-4 ( Operational amplifier )

TOPICS:- basic model; virtual ground concept; inverting amplifier; non-inverting amplifier; integrator; differentiator; Schmitt trigger; astablemultivibrator, Simple active filters: low pass, high pass, bandpass, notch filter

UNIT-5 ( Basic feedback theory )

TOPICS:- +ve and -ve feedback; concept of stability; oscillator, Waveform generator using op-amp schimitt trigger for Square wave, triangular wave Wien bridge oscillator for sinuisoidal waveform
